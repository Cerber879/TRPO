**Дата:** 20 сентября 2025 г.

**Статус:** Принято

**Контекст:**

Нужна интеграция с картографическим сервисом для отображения ресторанов на карте, построения маршрутов, геокодирования адресов и поиска ресторанов поблизости. Важно выбрать надежный и функциональный сервис, доступный в наших регионах.

**Рассмотренные варианты:**

1. **Yandex Maps API:** Российский картографический сервис с хорошим покрытием для России и стран СНГ. Поддерживает геокодирование, маршрутизацию, отображение карт.
2. **Google Maps API:** Международный стандарт с отличным покрытием по всему миру. Широкий функционал, но может быть дороже.
3. **OpenStreetMap / Leaflet:** Открытое решение с бесплатным использованием, но требует собственной инфраструктуры для геокодирования и маршрутизации.
4. **2GIS API:** Российский сервис с хорошим покрытием для крупных городов России, специализируется на бизнес-данных.

**Решение:**

Выбран Yandex Maps API как основной и единственный картографический сервис.

**Обоснование:**

- **Географический охват:** Мы планируем использовать наш сервис только на территории РФ, где Yandex Maps обеспечивает отличное покрытие и актуальные данные.
- **Надежность:** Yandex Maps не подвержен рискам блокировок и ограничений, которые могут коснуться Google Maps в России.
- **Стоимость:** Более выгодные тарифы для российского рынка по сравнению с Google Maps.
- **Функциональность:** Полный набор необходимых функций: геокодирование, обратное геокодирование, построение маршрутов, кластеризация меток.
- **Интеграция:** Хорошая документация и поддержка на русском языке, что упрощает разработку и решение проблем.
- **Опыт команды:** У нас уже есть опыт работы с Yandex Maps API, что ускорит интеграцию.

**Недостатки решения:**

- **Ограниченное международное покрытие:** Для расширения в другие страны потребуется дополнительная интеграция, но это не входит в наши планы на ближайшее время.
- **Зависимость от внешнего сервиса:** Необходимо обрабатывать случаи недоступности API и предусмотреть деградацию функциональности.

**Последствия:**

- Реализуем интеграцию с Yandex Maps API, используя наш опыт работы с этим сервисом.
- Настроим обработку ошибок и таймаутов при работе с внешним API.
- Организуем кэширование результатов геокодирования для снижения количества запросов к API и экономии средств.
- При недоступности картографического сервиса система должна продолжать работать с базовым поиском по адресам.
